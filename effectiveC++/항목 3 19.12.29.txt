1. const 위치에 따른 의미

2. const std::vector<int>::iteraton iter
   std::vector<int>::const_iterator iter
   차이점

3. 반환값을 참조자가 아닌 그냥 넘길 경우 해당 자료형의 사본이 넘어간다.

4. 상수 멤버 함수 
	해당 멤버 함수가 상수 객체에 대해 호출 될 함수이다. -> 해당 멤버 함수 안에서는 객체의 값이 바뀌지 않는다

5. 멤버 함수가 상수 멤버라는 것은 비트수준 상수성(못바꿈)과 논리적 상수성(사용자가 알아채지 못하게 바꿈)이 있다
	- mutable 키워드로 논리적 상수성을이용하서 프로그래밍 할수 있다

6. const의 유무로 함수 오버로딩이 가능하다
   만약 상수 멤버와 비상수 멤버 함수가 똑같이 구현되어 있으면 비상수 멤버가 상수 멤버를
   호출하게 만들자 -> cosnt_cast, static_cast
